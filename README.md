# learning-oauth2-server-client
おべんきょです。  
まずはコードで殴ってみる。

Passportの概念が特殊なのでそこの理解が必要そう。

## Requirements
- Node.js >= v8.0
- MongoDb
- Redis

## Setup

	$ npm install

## Usage
### OAuth2 Server
	$ npm run server
### OAuth2 Client
	$ npm run client
